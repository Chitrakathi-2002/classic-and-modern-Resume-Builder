# **Resume Builder with Advanced Features**

The **Resume Builder with Advanced Features** is an intuitive and customizable web application designed to help users create, edit, and preview resumes with ease. It offers a streamlined, user-friendly interface that allows individuals to quickly input personal details, professional experience, skills, and more. This tool also provides robust formatting options, allowing users to add custom sections, style their content, and select from multiple resume templates. Additionally, users can export their final resume in either Word or PDF format for easy sharing and printing.

### **Key Features:**

1. **Personal Information & Resume Sections**

   * Input and manage personal details such as name, job title, and contact information.
   * Add multiple sections like **Summary**, **Education**, **Experience**, **Skills**, **Projects**, and **Certifications**, or create custom sections based on the user's preferences.
   * Add profile photos with draggable placement for a personalized touch.

2. **Advanced Text Formatting**

   * Apply text formatting such as **bold**, **italic**, **underline**, and **font resizing** using a user-friendly toolbar.
   * Effortlessly undo or redo changes to maintain content accuracy.

3. **Customizable Resume Templates**

   * Choose from several pre-designed resume templates like **Default**, **Modern**, and **Classic** to give your resume a professional look.
   * Live preview of the resume, with real-time updates as users input data.

4. **File Export Options**

   * Export your resume as a **Word** document or **PDF** with customizable page size and margins, ensuring compatibility with different systems and professional settings.

5. **Data Persistence**

   * All user data, including entered content and profile photo, is automatically saved to **localStorage**, ensuring that users can resume work anytime without losing progress.

6. **Live Preview**

   * Real-time updates of the resume's appearance as content is added, allowing users to instantly visualize their resume in the selected template style.

7. **Drag-and-Drop Profile Photo**

   * Upload and position your profile photo anywhere on the resume preview, enhancing the customization and look of the final document.

### **User Flow:**

1. **Start by entering basic details**: Name, job title, and contact information.
2. **Add sections**: Select from predefined sections or create custom headings.
3. **Format the content**: Apply bold, italic, or underline to the text, adjust font size, and style it the way you want.
4. **Preview the resume**: Instantly see how your resume looks with live updates in the preview section.
5. **Export the resume**: Once you're happy with the final result, export the resume as a Word or PDF document.

### **Technologies Used:**

* **HTML5**: For building the structure and layout of the page.
* **CSS**: For styling and layout, with responsiveness for various screen sizes.
* **JavaScript**: For interactivity, such as saving content, updating the preview, and handling exports.
* **Local Storage**: For saving user data, so the progress is not lost.
* **Libraries**:

  * **FileSaver.js** for saving files locally.
  * **html2canvas** for generating a screenshot of the resume for PDF export.
  * **jsPDF** for generating PDF files.
  * **html-docx-js** for converting the HTML content to Word format.
